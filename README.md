# Simple Go Guestbook

Một ứng dụng web đơn giản viết bằng Golang, cho phép người dùng để lại lời nhắn (Guestbook). Dữ liệu được lưu trữ bền vững vào file JSON.

## Tính năng
- Gửi tin nhắn với Tên và Nội dung.
- Hiển thị danh sách tin nhắn mới nhất lên đầu.
- Lưu trữ dữ liệu vào file `data.json` (không mất dữ liệu khi tắt server).
- Giao diện HTML/CSS sạch sẽ, tách biệt.
